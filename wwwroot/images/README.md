# Images Folder

This folder contains static images used throughout the Holistic Department Exam System.

## Recommended Images:

### Logo and Branding
- `logo.png` - Main application logo (recommended size: 200x60px)
- `favicon.ico` - Browser favicon (32x32px or 16x16px)

### User Avatars
- `default-avatar.png` - Default user profile picture
- `admin-avatar.png` - Default admin avatar
- `student-avatar.png` - Default student avatar
- `coordinator-avatar.png` - Default coordinator avatar

### UI Elements
- `exam-icon.png` - Exam related icon
- `dashboard-bg.jpg` - Dashboard background image
- `login-bg.jpg` - Login page background

### Placeholder Images
You can add placeholder images for testing purposes:
- `placeholder-exam.jpg`
- `placeholder-user.jpg`
- `placeholder-result.jpg`

## Image Optimization Tips:
1. Use WebP format for better compression
2. Implement lazy loading for large images
3. Use appropriate image sizes for different viewports
4. Compress images without losing quality
5. Use CDN for production deployment
